=====================================================================================================
git status                       現在のリポジトリの状態を取得する．
git branch                       現在のブランチ名を取得する．
git add <FILE_NAMEs...>          リポジトリのインデックスに変更したファイルを指定して追加する．
git commit -m "MESSAGE"          インデックス内のファイルの変更をコミットする．
git push origin <BRANCH_NAME>    現在のブランチの内容をリモートリポジトリ（origin）に送る．
